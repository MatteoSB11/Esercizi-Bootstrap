Cos'è bootstrap?

Bootstrap è un framework front-end gratuito per uno sviluppo web più rapido e semplice.

Bootstrap include modelli di progettazione basati su HTML e CSS e ti dà anche la

possibilità di creare facilmente progetti reattivi.

Creazione di una pagina web con Bootstrap

< al posto del carattere < > al posto del carattere >

<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.3.1/css/bootstrap.min.css">

Cos'è un CDN?

Se non vuoi caricare e ospitare Bootstrap 5 da solo, puoi incuderlo da un CDN(Content Delivery Network)

jsDeliver fornisce un supporto per CSS e JavaScript di Bootstrap

A cosa serve il seguente attributo nel tag meta? content="width=device-width, initial-scale=1

La parte width=device-width imposta la larghezza della pagina in modo che segua la larghezza dello schermo del dispositivo

La parte initial-scale=1 imposta il livello di zoom iniziale quando la pagina viene caricata per la prima volta dal browser.

Che differenza c’è tra Container e Container Fluid?

La classe .container fornisce un contenitore reattivo a larghezza fissa

La classe .container-fluid fornisce un contenitore a larghezza intera, che copre l'intera larghezza del viewport
